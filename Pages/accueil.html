Mes Projets

Voici une présentation de mes projets.

Depuis le début de mes études, mon focus se retrouvait dans la performance de mes projets d'école mais principalement dans mon apprentissage personnel. Ma curiosité me pousse à toujours me dépasser et approfondir mes connaissances dans le domaine de l'informatique.

Puisque les projets font intégralement parti du développement de mes compétences, la qualité et l'efficience de ceux-ci se sont améliorés avec le temps.

Snake Game

Le jeu classique du "Snake Game".

En controlant un serpent, le but du jeu est de manger le plus de fruits possible avant de cogner un mur ou de se manger soi-meme.

Ce mini-programme est 100% codé en Javascript pour un résultat généré sur le web.

Répertoire GitHub

Memorize

Le jeu de mémoire "Memorize".

Un autre grand classique, il suffit de retourner les cartes et de rassembler les combinaisons de 2 emojis.

Ce mini-programme est 100% codé en Swift pour un résultat déployé sur toute plateforme IOS.

Répertoire GitHub

Soumission Assurance

Simulation d'une assurance vie.

Ce programme permet le calcul de la soumission d'une assurance vie selon différent critàres et retourne le résultat ainsi que le montant de celle-ci dans un fichier JSON.

Ce mini-programme est 100% codé en Java pour un résultat affiché sur un terminal.

Répertoire GitHub
